# Scripts

Hot-reloadable Rhai scripts that implement game logic.

| Directory | Purpose |
|-----------|---------|
| `commands/` | Player and builder commands |
| `data/` | JSON data files for character creation |
| `examples/` | Example area setups |
| `triggers/` | Reusable trigger scripts for rooms, items, and mobiles |

| File | Purpose |
|------|---------|
| `commands.json` | Command metadata (help text, permissions, aliases) |
