# Feature Implementation

2. After each file change, run `cargo build` to catch errors immediately
3. After all code changes, update tab completion in src/completion.rs for any new or renamed commands
4. Update relevant docs in docs/ if player-facing behavior changed
5. After all changes, run `cargo test`
6. Fix any failures before proceeding
7. Commit with a descriptive message and push
